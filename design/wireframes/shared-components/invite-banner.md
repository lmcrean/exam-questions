## Features

### For Teachers
- **Multiple Sharing Options**: Code, QR, email, direct link
- **Presentation Mode**: Full-screen for projecting
- **Copy to Clipboard**: One-click code sharing
- **Persistent Display**: Always visible when needed
- **Real-time Counter**: See how many students joined

### For Students
- **Simple Entry**: Just code or QR scan
- **Clear Instructions**: URL + code prominently displayed
- **Error Handling**: Helpful messages for invalid codes
- **Mobile Optimized**: Easy on phones/tablets
- **Quick Access**: Direct link option

### Technical
- **Auto-Generated Codes**: 6-character alphanumeric (uppercase)
- **Collision Avoidance**: Unique codes per session
- **QR Dynamic Generation**: Points to join URL with code
- **Responsive Design**: Works on all screen sizes
- **Accessibility**: High contrast, large text option
